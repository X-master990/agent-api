## Goal

The goal is to move from "local demo works" to "external people can understand, run, and evaluate the MVP."

## Beta Success Criteria

- 5 target users can understand the problem and demo.
- 3 users agree this addresses agent API access control.
- 2 users are willing to try it against a staging API or realistic mock.
- 1 user expresses willingness to continue into a PoC.

## Next Actions
